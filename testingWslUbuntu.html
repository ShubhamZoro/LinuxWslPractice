About this project

  • Earlier I used to use dual boot for running Windows and Linux simultaneously.
  • Since the introduction of WSL it's more convenient to use WSL instead of using Ubuntu dual boot with a powerful and more modern windows terminal.
  • This is just a practice project to get used to Ubuntu in WSL and to access Ubuntu file structure in windows.
  • Some of the useful commands are:
    • To access Ubuntu file structure open file explorer in windows and type "\\wsl$" in address bar.
    • To learn more about basic Ubuntu terminal commands click on this link Ubuntu Terminal Commands
